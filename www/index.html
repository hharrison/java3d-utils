This project contains the source code for the Java 3DTM core utilities in the com.sun.j3d.* hierarchy.

Related projects include: vecmath (the 3D vector math package), j3d-core (the core Java 3D API), and j3d-examples (the Java 3D example programs). The j3d-core-utils project is built as part of j3d-core. Developers should refer to the CVS download and build instructions for information on downloading the source code and building j3d-core-utils.

For all other project information, including contributing to Java 3D and reporting issues, please go to the parent java3d project.

NOTE: The issue Tracker in this j3d-core-utils subproject cannot be used to file or track issues. Please use the Issue Tracker in the parent java3d project to track issues (bugs, feature requests, etc.) for Java 3D-related sub-projects.
